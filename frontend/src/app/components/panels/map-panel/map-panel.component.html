Barents watch

This app was created to track Norwegian vessels. On the map there are circles in different colors which represent numbers of vessels in this region:

  • purple - 1-5 vessels
  • red - 6-10 vessels
  • yellow - 11-20 vessels
  • blue - above 20 vessels

At closed range vessels are treated separately (not in groups). This allows to check if the vessel position was registered more than 5 minutes ago (it is marked as grey circle, otherwise is green circle).

Logged userEntity can add single vessels to track them individually (having additional info about them).
